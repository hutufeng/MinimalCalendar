# 极简日历 (Minimal Calendar) 📅

一款基于 Android 极客精神，采用原生技术底座 (Kotlin + Jetpack Compose) 打造的纯粹、无广告、拥有极致性能的极简风格通用日历应用。

## ✨ 核心特性概览

- **极致流畅的双轴手势编排**：颠覆传统的翻页范式，支持极具质感的**上下顺滑滑动切换年份**，以及极其跟手的**左右滑动切换月份**，体验如丝般顺滑。
- **全硬核的离线能力基座**：依托开源历法引擎 (`lunar-java`)，完全剥离了对互联网的依赖，在任何纯黑无网的恶劣断网环境下，应用依旧瞬间吐出极其精准的公历、农历、廿四节气、干支纪年以及传统黄历的宜忌吉凶。
- **智能节假日多路追溯引擎**：抛弃极容易封杀受限的私人节点，利用全球多路 CDN 挂载 `NateScarlet/holiday-cn` 纯正 JSON 配置。独创内部**“4路 CDN 容灾降级轮询机制” (Fallback Retry)**，即便面临断网，也能依靠内置 DataStore 离线强缓展现当年国内复杂的休假和调休逻辑并在日期格高亮“休/班”，安全优雅降级无懈可击！
- **极致的 60FPS 渲染抗压**：大刀阔斧斩断了带来隐性测量负担的 `Lazy` 组件架构。纯静态组件配合“乐观 UI + 内存数据预挂载 + `Dispatchers.Default` 异步后台池剥离高密集算法” 的激进组合，斩杀一切滑动白屏与残影片段！
- **距今时光机**：选中任意一个未来的假期或者过往的时光，底部的悬浮面板自动帮你精准清点“距今还剩 X 天”或“已过 X 天”。

## 🛠️ 工程与选型信息

- **开发层核心**：Kotlin (1.9.22)
- **视觉层呈现**：Jetpack Compose (剥离 Material3 的花哨动态色，全定制的高反差极简主义灰阶映射色版)
- **业务层构架**：纯正的 ViewModel + MVVM 单向状态流体系
- **打包层强剪辑**：已在工程默认配置全面压码 `isMinifyEnabled` 与资源死区裁剪 `isShrinkResources`，将释放版本的体积极限压榨至仅仅约 **3MB**。
